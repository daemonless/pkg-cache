Daemonless pkg cache

FreeBSD pkg caching proxy — daemonless appliance

Configure a client

Save as /usr/local/etc/pkg/repos/FreeBSD.conf:

FreeBSD: {
  url: "http://<cache-host>/${ABI}/quarterly",
  mirror_type: "none",
  signature_type: "fingerprints",
  fingerprints: "/usr/share/keys/pkg",
  enabled: yes
}

The cache proxies whatever path the client requests — swap quarterly for latest to track rolling packages instead.
